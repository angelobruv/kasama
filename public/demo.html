Kasama

An open-source feedback layer. This whole page is wired with it.

Try it: select any text in this paragraph and a 💬 Comment button appears — leave a note anchored to exactly those words. Or hit ✎ Pin in the toolbar (bottom-left) and click anywhere to drop a numbered pin.

Comments thread, reply, and resolve. They sync to the server and persist — reload the page and they're still here, anchored to the same text.

That's the whole product: feedback that lives on the real page, not in a screenshot or a side doc. Self-host it, drop in one script tag, done.
